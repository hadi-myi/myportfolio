Social Network Web Application

Project Summary

This Social Network project is a web application built using Flask and SQLite, created as part of CS253. It allows users to register, log in, post updates, add friends, and view content from others in their network.

Core features include user authentication, the ability to create and browse posts, add friends, view user profiles, and search for other users by name or username.

Through this project, I learned how to build a full-stack web application using Flask, design relational databases, and implement basic social networking features from scratch.
