Python Chat Client

Project Summary

The Python Chat Client is a terminal-based messaging tool I built to explore real-time communication, networking, and secure protocols in Python.

It supports user authentication, open chat rooms, direct messaging, and file sharing between clients. All sent and received data is encrypted.

Users can register and log in, participate in public chat, send messages directly to others, and upload or download files from the server. All communication is handled over sockets using TCP.
